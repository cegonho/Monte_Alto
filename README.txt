
# Site multi-páginas — Parque de Autocaravanas

Estrutura baseada na navegação típica (Início, O Rancho, Serviços, Galeria, Novidades, Contactos).
**Não contém textos/imagens de terceiros.** Substitui pelos teus conteúdos.

## Onde colocar as imagens
- Pasta: `imagens/`
- Ficheiros hero por página: `hero-home.jpg`, `hero-o-rancho.jpg`, `hero-servicos.jpg`, `hero-galeria.jpg`, `hero-novidades.jpg`, `hero-contactos.jpg`
- Outras: `o-rancho-foto1.jpg`, `galeria1.jpg` ... `galeria6.jpg`

## Como usar
1. Abre `index.html` no navegador.
2. Edita textos em cada página HTML.
3. Coloca as tuas fotos na pasta `imagens/` com os nomes indicados (ou muda os `src` no HTML).
4. Publica em alojamento (FTP/cPanel) ou Netlify/Vercel.

## Nota legal
Cria textos e fotos originais. Evita copiar conteúdos de terceiros sem autorização.
